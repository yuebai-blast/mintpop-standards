# 设计基线

> **类型：`REFERENCE`（参考基线，可自定义）** · 适用范围：MintPop 品牌下所有产品
>
> 这是一套**默认起点**，不是铁律。各产品可按自身调性偏离——但偏离前先理解基线，保持品牌的整体气质（清爽、克制、现代）。**能改**：配色、字体、圆角、密度；**建议守**：整体的克制感与可访问性（对比度、可读性）。

## 一、配色（默认值，可覆盖）

以品牌薄荷绿为主色，产品可换主色但保留「浅色克制、强调色点缀」的思路。

| 语义 | 默认值 | 说明 |
|---|---|---|
| 主色 Brand | `#3DDC97` | 品牌薄荷绿；产品可替换为自己的主色 |
| 主色（深） | `#2BB47E` | hover / 按下态 |
| 文本 | `#1A1A1A` | 正文 |
| 次要文本 | `#6B7280` | 辅助信息 |
| 背景 | `#FFFFFF` / `#F7F8FA` | 页面 / 卡片分层 |
| 分隔线 | `#E5E7EB` | 边框、divider |

## 二、字体（REFERENCE）

- **无衬线优先**，系统字体栈起步，追求好看再自托管（**走 Fontsource 自托管，禁止外链 Google Fonts**，保证含中国大陆在内全球可达）。
- 默认字体栈：
  ```css
  font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC",
    "Microsoft YaHei", Roboto, "Helvetica Neue", Arial, sans-serif;
  ```

## 三、间距与圆角（REFERENCE）

- **间距刻度**：4 的倍数（4 / 8 / 12 / 16 / 24 / 32 / 48）。
- **圆角**：默认 `8px`（卡片）/ `6px`（按钮）/ `999px`（药丸标签）。产品可整体调大或调小，但保持一致。

## 四、可访问性（建议守住）

- 正文与背景对比度 ≥ 4.5:1（WCAG AA）。
- 交互元素有清晰的 hover / focus 态，focus 可见（键盘可达）。
- 不用「仅靠颜色」传达状态，配文字或图标。

## 五、每个产品可以自定义什么

| 项 | 能否改 | 说明 |
|---|---|---|
| 主色 / 强调色 | ✅ 可改 | 保留「克制、强调色点缀」的思路 |
| 字体 | ✅ 可改 | 但须自托管、全球可达 |
| 圆角 / 间距密度 | ✅ 可改 | 保持产品内部一致 |
| 品牌 Logo / 命名 | ❌ 不可改 | 见 [品牌规范](/brand/logo-and-usage)（INVARIANT） |
| 整体的克制感与可访问性 | ⚠️ 建议守 | 偏离要有明确理由 |
