# MintPop Game Seer

> 世界杯赛事预测工具（中文名「赛视」）。

概率由 Dixon-Coles 量化模型算出（可复现、加总 = 100%，不是 AI 现编），用户注入自己的信息优势（伤病、天气、战术判断），AI 分析师消化后输出研报格式的预测。技术栈 Nuxt 4 · Vue 3 + Tailwind v4。
